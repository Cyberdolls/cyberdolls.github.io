## Overview

AfridemiQ is a comprehensive learning and assessment platform designed specifically for African students. Our goal is to empower students to prepare for exams effectively through engaging quizzes and personalized progress tracking. This platform not only enhances learning but also builds confidence in students as they approach their examinations.

## Features

- **Personalized Learning Paths**: Tailor your study experience based on your strengths and weaknesses.
- **Interactive Quizzes**: Engage with quizzes that adapt to your learning pace.
- **Progress Tracking**: Monitor your performance over time to identify areas for improvement.
- **Community Support**: Join a community of learners to share tips and resources.
- **Multi-Device Compatibility**: Access the platform on mobile devices and desktops.
- **Offline Mode**: Download quizzes for offline access, making studying flexible and convenient.

## Technologies Used

- **Frontend**: React Native
- **Backend**: Node.js
- **Database**: MongoDB
- **Deployment**: AWS
- **Version Control**: GitHub

## Installation

To get started with AfridemiQ, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Cyberdolls/AfridemiQ.git
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd AfridemiQ
   ```

3. **Install Dependencies**:
   ```bash
   npm install
   ```

4. **Run the Application**:
   ```bash
   npm start
   ```

For the latest version and updates, check the [Releases section](https://github.com/Cyberdolls/AfridemiQ/releases).

## Usage

Once the application is running, you can:

- **Create an Account**: Sign up to start your learning journey.
- **Take Quizzes**: Choose from a variety of subjects and topics.
- **Track Your Progress**: View your results and adjust your study plan accordingly.

### Example Quiz Flow

1. **Select a Subject**: Choose a subject area you want to focus on.
2. **Start Quiz**: Begin the quiz and answer questions.
3. **Review Results**: After completing the quiz, review your answers and get instant feedback.
